# 商品管理後台系統

這是一個使用 Vite + React + React Router 建立的商品管理後台系統。

## 功能特色

- ✅ 用戶登入系統
- ✅ 路由保護（未登入用戶無法訪問商品頁面）
- ✅ 商品列表展示
- ✅ 商品詳細資訊頁面
- ✅ 響應式設計

## 安裝依賴

```bash
npm install
```

## 開發模式

```bash
npm run dev
```

啟動後，在瀏覽器中打開顯示的本地地址（通常是 http://localhost:5173）

## 使用說明

1. **登入頁面** (`/login`)
   - 輸入任意用戶名和密碼即可登入
   - 已登入用戶訪問登入頁會自動重定向到商品頁面

2. **商品列表頁面** (`/products`)
   - 顯示所有商品
   - 點擊商品卡片可查看詳細資訊
   - 可在此頁面登出

3. **商品詳情頁面** (`/products/:id`)
   - 顯示商品的完整資訊
   - 可返回商品列表

## 路由保護

- 未登入用戶嘗試訪問 `/products` 或 `/products/:id` 會被自動重定向到登入頁面
- 登入狀態使用 localStorage 保存

## 建置專案

```bash
npm run build
```

## 預覽建置結果

```bash
npm run preview
```

## 技術棧

- **Vite** - 建置工具
- **React 18** - UI 框架
- **React Router** - 路由管理
- **CSS3** - 樣式設計
